# Task 1: AI Website Copy Generator for Local Businesses
**Internship:** Future Interns (Prompt Engineering)  
**Client:** Apex AI Academy, Koormannapalem  

## 🚀 Project Overview
This project demonstrates a structured prompt engineering framework designed to generate conversion-focused website copy for local businesses.

## 🧠 Prompt Engineering Logic
* **Role Prompting:** Assigned the AI a persona of a "Senior AI Branding Specialist."
* **Contextual Injection:** Added specific geographical details (Koormannapalem/Vizag).
* **Negative Constraints:** Restricted the use of common "AI-isms" like "Welcome."

## 📁 Repository Structure
* `prompt.txt`: The master instructions (Source Code).
* `output.md`: The final website copy generated by the AI.
